# BattleShip

A multiplayer battleship game with AI opponent support, built with Haskell and JavaScript.

## Features

- Single player vs AI
- HTTP REST API
- Web-based client interface

### Backend (Server)

```bash
cd server
stack build
stack run
```

Server will start on `http://localhost:3000`

### Frontend (Client)

Open browser to: `http://localhost:3000`
